rfriends_chromeosはchromeOS上にrfriends3をインストールするスクリプトです。  
  
cd ~/  
sudo apt install git  
git clone https://github.com/rfriends/rfriends_chromeos.git  
cd rfriends_chromeos  
sh rfriends3_chromeos.sh
